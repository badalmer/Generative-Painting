FILE PREP
- .svg files must have compound paths removed.

DATA
- all vector files must be put in the DATA file

SETTINGS
density: number of layered elements
numColor: number of fill colors
fileNum: number of files in Data folder
line: outlines on/off
lineWidth: thickness of line (float)
BW: Greyscale on/off
